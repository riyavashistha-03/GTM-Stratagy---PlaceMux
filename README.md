# PlaceMux GTM Strategy Research

A self-initiated case study exploring the go-to-market strategy for a skill-based hiring platform.

## Objectives

- Evaluate competitive positioning
- Analyze customer pain points
- Design a B2B acquisition framework
- Identify monetization opportunities

## Deliverables

- Competitive benchmarking matrix
- Customer personas
- Product-market fit analysis
- B2B acquisition funnel
- Growth recommendations

## Key Insight

Traditional ATS systems prioritize resumes, while PlaceMux focuses on skill-based evaluation using L1-L100 scoring.
